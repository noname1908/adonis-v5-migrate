## Local development

Run the following to setup your local dev environment:

```sh
brew install node

# Clone or fork `ts-migrate`
git clone git@github.com:noname1908/adonis-v5-migrate.git # or your fork
cd adonis-v5-migrate

# install dependencies
npm i

# build packages
npm run build

# test packages
npm run test

# lint packages
npm run lint
```
